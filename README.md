# CS501 Applied Midterm Q1

## Counter Screen (Jetpack Compose)

This project implements a simple counter screen using Jetpack Compose.

## Features

1. Displays a number starting at **0**
2. Two buttons:
  - **Increment** → increases the count by 1
  - **Reset** → resets the count to 0
3. UI updates automatically when the state changes (recomposition)
4. State survives screen rotation using `rememberSaveable`

## Technologies Used

- Kotlin
- Jetpack Compose
- Android Studio

## Key Concepts

`mutableStateOf` for state management
`rememberSaveable` for preserving state across configuration changes
Compose UI and recomposition

## How to Run

1. Open the project in Android Studio
2. Run on an emulator or Android device
3. Use the buttons to interact with the counter
